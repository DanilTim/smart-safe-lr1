# SmartSafe — Лабораторная работа №1

Приложение-сейф для Android, который открывается секретной комбинацией наклонов устройства. Вместо пароля — физические движения телефоном.

---

## Функциональность

- **Запись комбинации** — пользователь наклоняет телефон в любую сторону (←→↑↓) от 3 до 8 раз, задавая уникальную последовательность
- **Разблокировка** — повторение сохранённой комбинации открывает сейф
- **Скользящее окно** — алгоритм проверки не требует точного старта ввода, достаточно правильно завершить последовательность
- **Защита от дребезга** — cooldown 600 мс между событиями сенсора исключает ложные срабатывания

## Запуск

1. Открыть проект в **Android Studio**
2. Подключить физическое Android-устройство **или** запустить эмулятор с поддержкой акселерометра
3. Нажать **Run ▶** (`Shift+F10`)

> ! Приложение требует наличия акселерометра (`android.hardware.sensor.accelerometer`). На устройствах без него будет отображено предупреждение.

---

## Как пользоваться

1. При первом запуске нажать кнопку **«Настройка»**
2. Наклонять телефон в разные стороны — записать комбинацию (минимум 3, максимум 8 наклонов)
3. Нажать **«Сохранить»** и вернуться на главный экран
4. Повторить сохранённую комбинацию наклонами — сейф откроется 🔓

## Параметры детектора

| Параметр | Значение | Описание |
|---|---|---|
| `threshold` | 6.0 м/с² | Порог ускорения для фиксации наклона |
| `cooldownMs` | 600 мс | Минимальный интервал между событиями |
| `MIN_COMBO_LENGTH` | 3 | Минимальная длина комбинации |
| `MAX_COMBO_LENGTH` | 8 | Максимальная длина комбинации |
